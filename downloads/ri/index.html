The JavaServer Faces Reference Implementation

This page provides links to download the binary and source distributions of the Reference Implementation (RI) for JSR 314 and JSR-252: JavaServer Faces, which represents a subset of the Mojarra project.

To utilize Mojarra distributions, the recommended approach to getting started and using JSF, please goto the Mojarra project page and click on the "Download Now" buttons.
